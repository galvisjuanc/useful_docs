Records para simplificar código inmutable en Java </h2>

<h3 align="left"> ¿Qué es un record en Java y para qué sirve? </h3>

<p align="left"> Los records son ideales para representar datos que no necesitas modificar nunca, evitando muchísimas líneas de código repetido y fomentando buenas prácticas modernas. </p>
